FAQ

Common questions

Everything you need to know before you start.

Is CyberGuard free to use?
Yes, completely free. The password analyzer, phishing detector, scam detector, and URL checker all run with no cost or account. The AI assistant requires a Groq or OpenAI API key to function.
Is my data stored or logged?
Content you analyse (emails, messages, URLs, passwords) is processed in memory and not persisted. Only anonymous tool usage counts are logged. Conversation history exists only in your browser session and clears when you close the tab.
How accurate are the security scanners?
The scanners use heuristic pattern matching — designed for educational awareness and first-pass triage, not as definitive security verdicts. They detect common red flags but cannot catch every sophisticated attack. Always exercise independent judgment and consult a professional for serious incidents.
Can CyberGuard tell me if a URL is definitely safe?
No tool can guarantee a URL is completely safe. The URL checker analyses structural indicators (suspicious TLDs, brand impersonation, URL shorteners, IP domains) but does not access the live URL or query real-time threat databases. Think of it as a pre-click sanity check.
Who is CyberGuard designed for?
CyberGuard is built for everyone — from individuals verifying a suspicious email, to security-aware professionals wanting a quick first-pass analysis. It provides beginner-friendly explanations while offering the depth experienced users expect.
Is this a replacement for professional cybersecurity services?
No. CyberGuard is an educational guidance tool. For active security incidents, data breaches affecting your organisation, or compliance matters, contact a qualified cybersecurity professional or your organisation's security team.
